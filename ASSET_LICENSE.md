# Tomb Crossing Asset License

Unless a file is explicitly marked otherwise, the original artwork, level
data, music, and sound effects in this repository are available under the MIT
License in `LICENSE`.

The GitHub asset archive includes `asset-manifest.json`. Each entry records
its path, byte size, SHA-256 digest, license, and release status:

- `active`: used by the official World 1-2 release.
- `coming_soon`: source material for World 3-5, not playable in production.
- `support`: shared source material.

Butano and its bundled third-party components are not project assets and are
not relicensed under MIT.
